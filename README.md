# React-Othello

![gif](https://github.com/kawasaki-n/react-othello/blob/main/public/app-image.png)

## Overview

オセロ（リバーシ）が遊べます。  

## Usage
 - 先手（黒）があなたで、後手（白）がコンピュータです。
 - 置ける場所がハイライトされます。

## Features
 - 置ける場所がない場合、自動的にパスされます。
 - 双方とも置ける場所がない場合、ゲームが終了し勝敗を判定します。
